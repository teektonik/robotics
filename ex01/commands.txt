1. pwd - показывает где я сейчас нахожусь
2. whereis - позволяет узнать где расположено приложение
3. history - показывает последние 1000 команд
4. df -h - показывает сколько места свободно на дисках, df -sh - показывает сколько весит директория
5. gzip - создает архив со сжатыми файлами
6. gunzip - распаковывает зип архив с расширением .gz и удаляет этот архив
7. uname -a - вся возможная системная информация (без -а только название ядра)
8. sleep - задает таймер выключения системы
9. time - позволяет подсчитать время исполнения процесса. real - общее время исполения, user - сколько времени
занял пользователь у CPU, sys - сколько времени GPU было потрачено системой
10. diff <file1> <file2> - производит точное сравнение двух текстовых файлов, выводя различие между ними
